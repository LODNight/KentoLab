# KentoLab - Tổng quan Dự án (Project Overview)

## 1. Ý tưởng cốt lõi (Core Vision)
**KentoLab** là một nền tảng học thuật tương tác (Educational Platform) chuyên sâu về Cấu trúc Dữ liệu và Thuật toán (Data Structures & Algorithms - DSA).
Mục tiêu của KentoLab là biến những khái niệm trừu tượng, khô khan thành các mô phỏng trực quan (Visualizations), giúp sinh viên/lập trình viên dễ dàng:
- Theo dõi từng bước thực thi (step-by-step) của một thuật toán.
- So sánh hiệu suất, hiểu rõ độ phức tạp (Time/Space Complexity).
- Đọc Pseudo-code song song với quá trình chạy thực tế.
- Tham khảo Source code thực tế bằng các ngôn ngữ như C++, Python.

## 2. Đối tượng mục tiêu (Target Audience)
- Sinh viên Công nghệ Thông tin đang học môn Cấu trúc Dữ liệu và Thuật toán.
- Những người tự học lập trình muốn ôn luyện kỹ năng giải quyết vấn đề (Problem Solving).
- Giáo viên/Giảng viên cần một công cụ trực quan để minh họa bài giảng trên lớp.

## 3. Các tính năng chính (Key Features)
1.  **Hệ thống Mô phỏng (Visualizer/Simulator):**
    - Hỗ trợ nhiều dạng dữ liệu: Mảng 1D (Sorting), Đồ thị (Graph), Cây (Tree), Bảng DP (Dynamic Programming).
    - Cung cấp Timeline (Thanh thời gian) cho phép Play, Pause, Tua ngược (Undo) hoặc Chạy từng bước (Step-by-step).
2.  **Từ điển Học thuật (Algorithm Encyclopedia):**
    - Chứa đầy đủ Meta-data: Mô tả khái niệm, điều kiện áp dụng, ứng dụng thực tế.
    - Code Snippets (C++, Python).
3.  **Hệ thống User (Auth & Profile):**
    - Đăng nhập/Đăng ký tài khoản (hỗ trợ giả lập qua localStorage).
    - Quản lý hồ sơ cá nhân.
4.  **Hệ thống Gói cước (Payment/VIP):**
    - Mô phỏng thanh toán nâng cấp tài khoản từ Free lên Pro/Enterprise.
    - Mở khóa các thuật toán độc quyền (VIP Algorithms).

## 4. Định hướng tương lai (Future Roadmap)
- Bổ sung hệ thống Challenge/Problem: Giao bài tập để user viết code và chấm điểm trực tiếp.
- Ra mắt phiên bản Mobile/Responsive hoàn chỉnh cho hệ thống Visualization.
- Backend hoá toàn bộ hệ thống bằng Node.js + Database thực (MongoDB/PostgreSQL).
